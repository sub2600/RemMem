
Victorian Parliament Memory Tool – Admin Guide

1️⃣ CSV Upload Method
- Prepare a CSV file with columns: name, party, electorate, image
- Go to Admin Dashboard → CSV Upload
- Select and submit the file

2️⃣ Bulk Image + Metadata Upload Method
- Prepare images and a members.json file with member details
- Go to Admin Dashboard → Bulk Upload
- Select and submit the files

3️⃣ Manual Form Entry Method
- Go to Admin Dashboard → Manual Entry
- Fill out the form and submit

Styling:
- Dark blue background with light text
- Styled buttons and cards for a modern look
